The file: EnsemblePhishingDetection.py - is the software that was used to carry out the experiments. To run this program you will need Python 3 and the various libraries imported at the top of the file. 
The folder: Datasets - contains Dataset's 1 and 2 as referenced in the dissertation.
The files: scoring_results_dataset=1.csv and scoring_results_dataset=2.csv - these contain the raw output result from the experimental software. 
The folder: Graphs - is where the ROC curves are saved from the experimental software.
